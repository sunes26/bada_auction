# Playauto 카테고리 설정 필요

## 문제 상황

상품 등록 시 다음 오류 발생:
```
플레이오토 API 오류: 지원하지 않는 카테고리입니다.(1)
error_code: e4014
```

## 원인

1. **sol_cate_no는 필수 필드**입니다
2. `/api/categorys` API 조회 결과 카테고리가 **0개**로 나옴
3. 플레이오토 계정에 **카테고리가 아직 설정되지 않음**

## 해결 방법

### 옵션 1: 플레이오토 웹에서 카테고리 설정 (권장)

1. https://playauto.io 에 로그인
2. 설정 > 카테고리 관리로 이동
3. 사용할 쇼핑몰별 카테고리 매핑 설정
4. 설정 완료 후 `/api/categorys` API로 조회하여 `sol_cate_no` 확인

### 옵션 2: 상품별 카테고리 수동 지정

데이터베이스에 상품 등록 시 `sol_cate_no` 컬럼에 플레이오토에서 확인한 카테고리 번호 직접 입력

## 참고

- 카테고리 조회 API: `GET /api/categorys`
- 파라미터: `start=0, length=1000` (최대 1000개)
- 플레이오토 문서: "카테고리 조회 API를 통해 확인된 번호 입력"

## 테스트 결과

```bash
# 카테고리 조회 시도
GET /api/categorys?start=0&length=1000
응답: []  # 빈 배열

# 상품 등록 시도 (sol_cate_no: 11354)
POST /products/add/v1.2
오류: e4014 - 지원하지 않는 카테고리입니다
```

2. 설정 완료 후 API로 카테고리 목록 조회
3. 조회된 `sol_cate_no` 값을 상품 등록 시 사용
